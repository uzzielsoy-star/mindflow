🔗 Compartir Tablero

Exporta como JSON para compartir o guardar tu tablero. Para sync en tiempo real en la nube, integra Firebase Realtime Database con el formato exportado.

📥 Importar desde Miro

Importa un tablero de Miro vía API. Solo necesitas el ID del tablero y un token de acceso.

1
Obtén tu token de Miro
Ve a miro.com → Perfil → Apps, crea una app con permiso boards:read y genera un token de acceso.
2
Copia el ID del tablero
En la URL de tu tablero: miro.com/app/board/uXjVPiJoHnA=/
— o importa un JSON exportado de la API —
